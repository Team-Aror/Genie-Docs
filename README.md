# 🚀 KnowGenie AI — Intelligent Document Assistant

KnowGenie AI is an AI-powered document intelligence platform designed to simplify document understanding using advanced AI models.

It provides:

- 📄 Smart Document Summarization  
- ❓ Automatic FAQ Generation  
- 💬 AI Chat Assistant  
- 🔎 (Upcoming) RAG-based Document Retrieval System  

---

## 🌐 Live Project

- **Frontend URL:** _genie-docs-six.vercel.app_
 

---

# 🛠 How to Run This Project Locally

## ✅ Requirements

- Node.js (v18+ recommended)
- npm

---

## 📌 Setup Instructions

```bash
# Clone repository
git clone <[YOUR_GIT_URL](https://github.com/Team-Aror/Genie-Docs)>

# Navigate into folder
cd <Genei-Docs>

# Install dependencies
npm install

# Start development server
npm run dev
````

App runs on:

```
http://localhost:5173
```

---

# 🔐 Environment Variables

Create a `.env` file in the root directory:

```env
VITE_GEMINI_API_KEY=your_gemini_api_key_here
```

⚠️ Do not upload `.env` to GitHub.
